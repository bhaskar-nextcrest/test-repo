Integrations

EmailGmail Integration

Integrates with Gmail to send and receive emails, trigger workflows based on new emails, and automate email-related tasks.

Auth: OAuth 2.0
Endpoints: gmail.googleapis.com/v1/users/{userId}/messages/send gmail.googleapis.com/v1/users/{userId}/messages

Integrates with Hubspot CRM to manage contacts, create deals, and automate marketing and sales workflows.

Auth: API Key
Endpoints: api.hubapi.com/crm/v3/objects/contacts api.hubapi.com/crm/v3/objects/deals

Auth: OAuth 2.0

Automation Pipelines

New Email to CRM Contact (trigger: New email received in Gmail)
  1. Extract sender information from the email
  2. Search for the contact in Hubspot CRM
  3. If the contact exists, update the contact's information
  4. If the contact does not exist, create a new contact in Hubspot CRM
Schedule Event and Send Email (trigger: User command via chat to schedule an event)
  1. Parse event details (date, time, description) from user input
  2. Create event in Google Calendar
  3. Send a confirmation email to the attendees via Gmail
